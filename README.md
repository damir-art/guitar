### Бой №2 на гитаре
Песни: Кукушка (Цой), Как на войне (Агата Кристи), Кукла колдуна (КиШ).

![Описание](/img/boy-1.png "Подпись")

`Am  C  G  G` - 4р/2р проигрыш

Куплет 2 раза:

    Am  Am  C   C
    G   Dm  Am  Am

Припев 3 раза:

    G  Dm  Am  Am
    G  Dm

Вместо `Dm` у припева можно использовать `F`.
